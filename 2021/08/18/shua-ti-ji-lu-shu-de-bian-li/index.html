刷题记录：树的遍历

发表于 2021-08-18

一. 深度优先

前中后三种遍历

  1. 递归

    树是一个很经典的递归的结构，因为每个节点都可以看作根节点；

    递归有三个重点：递归函数的参数和返回值；终止条件；单层逻辑
    对二叉树来说，访问的顺序一定是先根节点，再到左孩子和右孩子
    所以这里的前序、中序、后序其实说的是对根节点所做操作（比如遍历就是将根节点的值加入到结果中）的顺序

下面就分析一下遍历二叉树的递归三要素：

  • 递归函数的参数和返回值：
    • 参数：根节点root，保存遍历结果的list
    • 返回值：void
  • 终止条件：当前根节点为空
  • 单层逻辑：每一层要做三件事：（1）保存当前根结点的值到结果，递归处理根节点的左子树，递归处理根节点的右子树；依据这三件事的处理顺序就有了前序中序后序
    • 前序 保存根节点的值–>处理左子树–>处理右子树
    • 中序：处理左子树–>保存根节点的值–>处理右子树
    • 后序：处理左子树–>处理右子树–>保存根节点的值

理解了逻辑以后，很容易写出递归的代码

//前序

递归可以理解为一个不断压栈的过程：递归也是一种函数调用；在发生函数调用时，操作系统会在栈中为被调用的函数开辟一块空间，存储所需要的数据和返回地址
函数调用的主要步骤：

  1. 参数入栈：将参数从右到左压入系统栈中
  2. 返回地址入栈：将当前调用指令的下一条指令地址入栈，供函数调用结束后继续执行后面的部分
  3. 代码区跳转：处理器处理被调用函数的代码部分
  4. 保存当前栈帧状态:EBP入栈
  5. 更新栈帧：将ESP的值拷贝到EBP
  6. 更新ESP

考虑一个向下增长的栈：其中EBP总是指向当前栈帧的栈底，ESP指向当前栈帧的栈顶，EIP为指令寄存器，存放一个指针，该指针永远指向下一条待执行的指令地址

  1. 迭代：迭代其实就是自己用一个栈模拟递归压栈的过程
    • 根据栈后入先出的特征:我们希望得到前序遍历的顺序是：root->left->right,那入栈顺序应该是，right,left,root;中序顺序应该是：right,root,left;后序顺序应该是：root,right,left;
    • 为了统一处理，我们每次从栈中弹出一个节点，以当前节点作为root节点，再按照遍历顺序将root,left,right入栈，这样root节点会先出栈再入栈，我们只需要对它做标记(加一个null做分隔符），第一次出栈的作用是找到当前根节点，以处理它的左孩子和右孩子；只在它第二次出栈时，才将它的值加到结果中
  //前序
  public void preOrder(Treenode root,List<Integer>res)&#123;
      Stack<Treenode> stack=new Stack<>();
      List<Integer> res=new ArrayList<>();
      stack.push(root);
      while(!stack.empty())&#123;
          Treenode node=stack.pop();
          if(node!=null)&#123;//第一次出栈
            if(node.right!=null)stack.push(node.right);
            if(node.leftt!=null)stack.push(node.left);
            stack.push(node);
            stack.push(null);//添加分隔符，标记这是第二次入栈
          &#125;
          else&#123;//第二次出栈，要将值加到结果中
            Treenode cur=stack.pop();
            res.add(cur.val);
          &#125;
      &#125;
      return res;//栈空时表示遍历结束

广度优先

层次遍历

层次遍历的逻辑比较简单，借助队列，每次取出队首元素时，做三件事：左孩子入队，右孩子入队，将当前节点的值加入结果

下面总结了Leetcode上几道层次遍历的题，根据题意的要求在一些小细节上需要不同的处理

  1. 102：普通层次遍历

    每层的结果单独为一个列表，所以在处理前首先要判断当前队列中的元素数量
   public List<List<Integer>> levelOrder(TreeNode root) &#123;
        Queue<TreeNode> queue=new ArrayDeque<>();
        List<List<Integer>> res=new ArrayList<>();
        if(root==null)return res;
        queue.add(root);
        while(!queue.isEmpty())&#123;
            List<Integer> cur=new ArrayList<>();//保存当前层的结果
            int n=queue.size();
            for(int i=0;i<n;i++)&#123;
                TreeNode node=queue.poll();
                cur.add(node.val);
                if(node.left!=null) queue.add(node.left);
                if(node.right!=null) queue.add(node.right);
            &#125;
            res.add(cur);
        &#125;
        return res;
    &#125;

    自底向上，只需要改变每一层的结果的插入位置
    public List<List<Integer>> levelOrderBottom(TreeNode root) &#123;
        Queue<TreeNode> queue=new ArrayDeque<>();
        LinkedList<List<Integer>> res=new LinkedList<>();//双向链表
        if(root==null) return res;
        queue.add(root);

        &#125;

  1. 103. 二叉树的锯齿形层序遍历

    每一层保存结果时需要判断是从左到右还是从右到左
   public List<List<Integer>> zigzagLevelOrder(TreeNode root) &#123;
        Queue<TreeNode> queue=new LinkedList<>();
        List<List<Integer>> res=new ArrayList<>();
        if(root==null)return res;
        queue.add(root);
        boolean right=false;//为true表示从右到左
        while(!queue.isEmpty())&#123;
            int n=queue.size();
            LinkedList<Integer> cur=new LinkedList<>();//实现双向插入
            for(int i=0;i<n;i++)&#123;
                TreeNode node=queue.poll();
                if(node.left!=null) queue.add(node.left);
                if(node.right!=null) queue.add(node.right);
                if(right) cur.addFirst(node.val);
                else cur.addLast(node.val);    
        &#125;
        right=!right;
        res.add(cur);
    &#125;
    return res;
    &#125;
